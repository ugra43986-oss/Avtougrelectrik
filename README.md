## Как редактировать сайт

Чтобы изменить текст сайта:

1. Открой файл `index.html`
2. Нажми **Edit**
3. Внеси изменения
4. Нажми **Commit changes**

Чтобы заменить главное фото:

1. Загрузи новый файл вместо `hero-service.jpg`
2. Если имя другое — измени путь к фото в `index.html`

## Примечание

Если изменения не появились сразу:

- подожди 1–2 минуты
- открой сайт заново
- нажми **Ctrl+F5** для полного обновления страницы
